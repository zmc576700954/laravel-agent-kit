本目录用于将不同 AI 工具的“可直接引入产物”拆分为独立包，便于以 git subtree/镜像仓库方式单独分发。

- `packages/claude/`：Claude Code 入口包（根目录 `CLAUDE.md`）
- `packages/codex/`：Codex 入口包（根目录 `AGENTS.md`）
- `packages/opencode/`：OpenCode 入口包（根目录 `AGENTS.md`）
- `packages/trae/`：Trae skills 包（`skills/<skill>/SKILL.md`）
- `packages/qoder/`：Qoder 项目内包（根目录 `.qoder/**`）
- `packages/project-assets/`：可选脚本/模板资产包
