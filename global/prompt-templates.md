# Laravel 任务 Prompt 模板（通用）

## 使用说明

- 按模板填写信息后再发给 Agent；不确定的字段用 `UNKNOWN` 标记。
- Agent 必须逐项回应，不得跳过“约束 / 验收 / 验证”。
- Agent 输出必须包含：范围控制 → 任务分级 → 计划/实施/验证闭环（Scope / Task Level / Plan / Implement / Verify）。
- 若涉及新增依赖、鉴权模型调整、生产配置、接口契约变更、中文正文文件写入或数据迁移策略变更，Agent 必须先停下追问确认。
- 若任务属于项目迁移批次或需要项目特有规则，请在模板中明确启用 Profile（默认不启用）。
